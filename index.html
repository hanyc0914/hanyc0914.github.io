树状数组（BIT）

概念

维基百科
树状数组求逆序数
lowbit(x) 函数：lowbit(x) = x & -x，表示 x 用二进制表示的最后一个 1 所在的位置表示的数值。例如 6 = 110，lowbit(6) = 10，也就是 2。
A[n]：长度为 n 的数组。
BIT[n+1]：长度为 n + 1 的数组，实际有效下标从 1 开始，用于维护 A[n] 数组的前缀和。BIT[k] 表示从 A[k] 开始往前的 lowbit(k) 个数的和，也就是 A[k - lowbit(k) + 1] + … + A[k]。

leetcode 315. Count of Smaller Numbers After Self

题目：给定一个数组 nums[n]，返回一个新的数组 counts[n]，其中 counts[i] 为 nums[i] 右边比它小的数的个数。
例子：

Input: [5,2,6,1]
Output: [2,1,1,0]
Explanation:
To the right of 5 there are 2 smaller elements (2 and 1).
To the right of 2 there is only 1 smaller element (1).
To the right of 6 there is 1 smaller element (1).
To the right of 1 there is 0 smaller element.

分析1：可以利用 BIT 求和的思想来求比 x 小的数字个数，只需要把原来存的前缀和改成1，也就是在 update 时把更新前缀和改为对应元素加一，那么 BIT[k] 表示前 k 个数中比 nums[k] 小的的个数。在遍历时从后往前遍历，转化为求比当前数字小的个数。另外一般定义下 BIT 的下标是数组中的元素，也就是需要开一个很大的 BIT 数组保证原数组中的最大元素有位置放，因此可以采用离散化的方式来存储原数组的信息：先将原数组排序，从小到大依次填充到 map 中，这样遍历到 nums[i] 的时候可以直接按它的排序下标来更新，也就是 [10,20,500,1000] -> [1,2,3,4]，遍历到 1000 时按 4 更新，减少开辟空间。

分析2：可以用归并排序，利用归并排序在合并左右数组时比大小，来更新 nums[i] 右边小的数的个数。从后往前遍历左右数组，当左数组的当前元素 i 大于右数组的当前元素 j 时，由于左右数组有序，说明右数组中 (m + 1, …, j) 这几个数都比 i 小，对左数组元素计数。

leetcode 82. Remove Duplicates from Sorted List II

题目：给定一个有序的链表，其中有些数字重复，返回一个新链表（只保留出现一次的数字）
例子：

Input: 1->2->3->3->4->4->5
Output: 1->2->5

分析：使用两个指针，prev 表示上一次保留的数字，p 表示当前正在判断的数字。当 p 有效时进入以下循环：1.先判断 p 和 p->next 是否相同，如果相同则将 p 指向下一位继续判断（循环），如果不同进入下一步骤。2.如果 p 是移动过的，说明当前的 p 仍然指向重复过的数字，因此将 p 后移一位，回到步骤 1 重新进入循环。3.如果 p 没有移动过，将 prev 指向 p，回到步骤 1 判断后续元素。4.循环结束后，需要将 prev->next 置为 1，否则当最后一个元素是重复元素时，会将这些元素加到新链表。

leetcode 792. Number of Matching Subsequences

题目：给一个字符串 S 和 和一个词典列表 words，找出 words[i] 的个数（是 S 的子序列，不需要连续）

例子：

Input: 
S = "abcde"
words = ["a", "bb", "acd", "ace"]
Output: 3
Explanation: There are three words in words that are a subsequence of S: "a", "acd", "ace".

分析：使用字典记录单词，先遍历 words 以每个单词的首字母建立一个 word_dict，每个字母的 value 是对应首字母的 word 列表；然后遍历 S 的字符 s，找到以 s 为首字母的单词列表去遍历单词列表，每次访问到一个单词就从列表中移除，然后判断是否检查到最后一个元素，如果是则计数加一，否则去掉单词首字母将新单词加入到 word_dict 中。

注意：用 python 实现时发现如果 word_list = word_dict.get(s)，并且后续在 word_dict 增加元素，word_list 也会改变，所以不能遍历 word_list，而是需要遍历长度；python 遍历英文字母不能像 c++ 一样 ‘a’ + 1

滑动窗口

一般用双端队列 deque 或双指针来记录窗口情况

leetcode 239. Sliding Window Maximum

题目：给定一个数组 nums，数字 k 表示窗口大小，窗口从下标 0 开始移动，输出窗口每次停留时的最大值

例子：

Input: nums = [1,3,-1,-3,5,3,6,7], and k = 3
Output: [3,3,5,5,6,7]

分析：使用双端队列 deque，队首表示当前窗口内值最大的元素的下标，从队首到队尾的值依次减小。每次访问到一个元素，判断对首下标是否超出窗口（==i-k），如果超出则删除，然后依次删除比当前元素小的队尾元素，然后再加入到队列中。

leetcode 3. Longest Substring Without Repeating Characters

题目：给定一个 string，找出最长连续不重复子串

例子：

Input: "abcabcbb"
Output: 3 
Explanation: The answer is "abc", with the length of 3.

分析：使用 l 和 r 两个坐标分别表示子串的初始位置和终止位置，建立一个字典记录子串中字母出现的次数。从左往右开始遍历，如果当前元素没出现过，则记录频数坐标右移；如果出现过或遍历到字符串最后一位，则初始位置的元素取出，起始坐标后移一位。注意，r 表示当前判断元素，所以出现重复或遍历完会重新进入循环再次判断 r 的元素。

leetcodde 209. Minimum Size Subarray Sum

例子：

Input: s = 7, nums = [2,3,1,2,4,3]

Explanation: the subarray [4,3] has the minimal length under the problem constraint.
